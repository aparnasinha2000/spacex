# spacex
To run the app</br>

1. Clone the repository on your local system : </br>
  git clone https://github.com/aparnasinha2000/spacex.git</br>

2. In the global environment, install dependencies :</br> 
   npm install</br>

3. Run the app on local host :</br> 
   npm start
